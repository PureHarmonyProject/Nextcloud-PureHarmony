# Nextcloud Client for HarmonyOS

This is a EXTREMELY early version of Nextcloud Client on HarmonyOS.

Supported Features:

- Logging in
- Listing Files
- Navigating Among Folders
- Uploading Photos
- Previewing Photos/Files that has a preview
